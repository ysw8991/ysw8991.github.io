You can not run away (1)

国庆节结束后过去四周零两天了，也就是三十天。

学习上遇到了不少麻烦。第一周的时候，看Network Analysis这本书很有动力。可是越到后来，越发感到读这本书的痛苦。现在只希望老师能尽快分配新的任务。大型应用软件设计这门课完全跟不上队长的进度，只好放弃，只希望最后的成绩不要太难看。计算机图形学的作业写的很是应付。本来计划好好学习一下OpenGL，可是学习的成本太高，可能会花很多时间，就也放弃了。准备六级也越来越没有动力。

不过学习上还是有一些小成果的。比如顺利完成了Linux原理与应用这门课的大作业，还是班里第一个完成的。

生活上，遇到了一个感觉很好的人，希望能和TA有更深的了解。前天晚上和LLJ聊了会儿天。快两年没见到他了，寒假回家一定要抽时间和LLJ一起出来玩玩。

娱乐方面，看了不少的动画、电影。观感最好的是汤浅政明的恶魔人 Crybaby、乒乓和心理游戏。黑泽明的电影没想象中那么好看，可能是不合我口味吧。还看了杨德昌的麻将。杨德昌的电影实在是有些冷酷了。接下来准备把杨德昌剩下的几部电影也给看了。完全没看书。双十一应该买些书。

最近最大的痛苦就是突然间想去保研，因为想去上学硕。可是因为成绩太差，感觉连推免资格都不一定能拿到。现在想去参加一些项目来丰富简历也很难了。好好把实验室的任务做好吧。其他的事情走一步看一步吧。

其实自己完全可以放弃名校情结，在本校读研也是一个不错的选择。毕竟读名校和人生的幸福也没有必要的联系。可是还是想去更牛的学校啊。自己都被自己给搞糊涂了。麻将中有这样一句话：““现在的世界没有人真正知道自己想要什么，而是要别人来告诉他该怎么做。” 蛮符合自己现在的心境的。

总之，还是很迷茫，不知道要做什么好。想起了FLCL中真见美的香烟上的字：”NEVER KNOWS BEST”。但是，请好好学习，认真生活，找到自己的答案。

You can not run away.
